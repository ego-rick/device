Окно "Работа с базой данных"
============================

Раздел 1. Описание окна.
Раздел 2. Порядок работы.

============================



РАЗДЕЛ 1. ОПИСАНИЕ ОКНА
-----------------------------------------------------------------------------------------------------

Активные поля окна:
	- "Выполняет" - ввод данных выполняющего операцию.
	- "Дата" - ввод даты.
	- "Время" - ввод времени.
	- "Примечание" - ввод примечания.

СТРОКА "ТЕКУЩАЯ ОПЕРАЦИЯ"...
Содержит в себе 2 неактивных поля (номер операции и её название) и кнопку завершения операции.

СРОКА "РЕМОНТ"...
Содержит в себе кнопку начала ремонта, выпадающий список с номером отката операции, неактивное поле
с названием операции отката, кнопку завершения операции ремонта.

ОСТАЛЬНЫЕ ЭЛЕМЕНТЫ:
	- Кнопка "Вывести список завершенных операций" - открывает окно просмотора завершенных операций 
по прибору.
	- Кнопка "Сохранить данные" - сохраняет вводимые данные (поля: "Выполняет", "Дата", "Время", 
"Примечание", выпадающий список отката опрации) по текущей операции.
	- Кнопка "Назад" - вернет в окно выбора прибора.
	- Кнопка "Выход" - вернет в окно авторизации.



РАЗДЕЛ 2. ПОРЯДОК РАБОТЫ.
-----------------------------------------------------------------------------------------------------

Схема:
Заполнение данных -> подтверждение.

Описание:
1. В открытом окне заполнить ключевые поля:
	- "Выполняет";
	- "Дата";
	- "Время";
	- "Примечание".

	По окончанию дейстивий нажать "Сохранить данные".
	При внесении поправок в данные текущей операции также нажать "Сохранить данные".

2. Завершение операции.
	Перед завершением операции убедиться, что все ключевые поля заполнены!
	В строке "ТЕКУЩАЯ ОПЕРАЦИЯ" нажать кнопку "Завершить".

3. Ремонт.
	Перед переходом в режим ремонта убедиться, что все ключевые поля текущей операции заполнены!
	Данные операции будут сохранены, после чего создастся отдельная запись с данными по ремонту.
	

	Нажать кнопку "Ремонт".
	В выпадающем спсике строки "РЕМОНТ" выбрать номер отката операции.
	Ввести основные данные и сохранить их.


	Перед завершением операции убедиться, что все ключевые поля заполнены!
	В строке "РЕМОНТ" нажать кнопку "Завершить".
