Описание программы
=====================

Программа представляет собой многооконное приложение с двумя режимами работы: инженер, начальник.
Описание окон программы:
1.	«Авторизация» — окно входа в программу.
2.	«Настройка база данных» — окно настройки подключения к базе данных.
3.	«Начальник — отображение операций» — окно вывода информации по прибору для начальника.
4.	«Инженер — список завершения операций» — окно вывода всех операций по прибору.
5.	«Инженер — выбор прибора» — окно выбора прибора для последующей работы.
6.	«Окно создания нового прибора» — окно для регистрации нового прибора в базе данных.
7.	«Инженер — работа с базой данных» — окно работы с прибором.

Порядок работы с программой для начальника:
1.	Запуск программы.
2.	Авторизоваться в окне «Авторизация» (при потребности подключение базы данных в окне «Настройка база данных» вызываемое кнопкой «Database»).
3.	В открывшемся окне «Начальник — отображение операций» начать работу.
    a.	В выпадающем списке «Тип прибора» выбрать соответствующие данные.
    b.	В выпадающем списке «№ прибора» выбирать соответствующие данные.
    c.	Нажать «Загрузить».

Порядок работы с программой для инженера:
1.	Запуск программы.
2.	Авторизоваться в окне «Авторизация» (при потребности подключение базы данных в окне «Настройка база данных» вызываемое кнопкой «Database»).
3.	Выбор прибора в окне «Инженер — выбор прибора»:
    a.	Выбор существующего прибора: в таблице «список приборов» выделить прибор путём левого клика мыши, нажать кнопку «Выбрать прибор».
    b.	Регистрация нового прибора: нажать кнопку «Новый прибор», в открытом окне ввести данные и нажать «Создать», далее аналогично для выбора существующего прибора.
4.	В открывшемся окне «Инженер — работа с базой данных» начать работу.

Описание работы в окне «Инженер — работа с базой данных»:

Если прибор не на ремонте
1.	В открытом окне заполнить ключевые поля:
    a.	"Выполняет";
    b.	"Дата";
    c.	"Время";
    d.	"Примечание".
2.	При внесении поправок в данные текущей операции также нажать "Сохранить данные". По окончанию действий нажать "Сохранить данные".
3.	Для завершения операции убедиться, что все ключевые поля заполнены. В разделе "ТЕКУЩАЯ ОПЕРАЦИЯ" нажать "Завершить".

Если прибор на ремонте
1.	Перед переходом в режим ремонта убедиться, что все ключевые поля текущей операции заполнены. Данные операции будут сохранены, после чего создастся отдельная запись с данными по ремонту. Нажать кнопку "Ремонт"
2.	В выпадающем списке строки "РЕМОНТ" выбрать номер отката операции.
3.	Ввести основные данные и сохранить их.
4.	Для завершения операции убедиться, что все ключевые поля заполнены. В разделе "ТЕКУЩАЯ ОПЕРАЦИЯ" нажать "Завершить".
