Порядок работы:
===============
Авторизация:
	1. Подключить БД (кнопка "Database");
	2. Авторизироваться.

Начальник:
	1. Выбрать тип и номер прибора;
	2. Нажать "Загрузить".

Инженер:
	1. При надобности создать прибор:
		- Нажать "Новый прибор";
		- Заполнить все поля и нажать "Создать".
	2. Выбрать прибор:
		- В списке нажать на требуемый;
		- Нажать на кнопку "Выбрать прибор".
	3. Работа в поледующем окне описана в подсказке.
		- Кпока "?" по левую сторону заголовка окна (файл ".\infoEngWork.txt").

Окно отображения всех операций:
	Сохранение в WORD (примитивный уровень):
	- Убедиться, что в директории не находится файл ".\table.docx"
	- Нажать на кнопку "W" по левую сторону заголовка окна.
